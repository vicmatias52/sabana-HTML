Etiquetas extra

< details> y < summary> Estas etiquetas permiten crear un componente interactivo que puede expandirse o colapsarse para mostrar u ocultar información adicional.


La etiqueta < abbr> se utiliza para definir una abreviatura o un acrónimo. Al pasar el ratón por encima, se muestra el significado completo. ok

< mark> Resalta texto en la página, generalmente con un fondo amarillo, similar a un marcador de resaltado.

laalalakalal



< meter> Muestra una barra de progreso que representa una medida dentro de un rango conocido. 70%

< progress> Similar a `< meter>`, pero se usa para mostrar el progreso de una tarea en ejecución. 32%

< cite> Se utiliza para citar el título de una obra como libros, películas, etc.

Un ejemplo clásico es Don Quijote de la Mancha de Miguel de Cervantes.
