Etiquetas HTML (estructura basica)

  1. "!DOCTYPE":

    Se utiliza para especificar la versión de HTML que se está utilizando en la página.

  2. "html": Define el inicio y el final de la página web.
  3. "head": Define la sección de encabezado de la página, donde se incluyen elementos como el título de la página, metaetiquetas, scripts, entre otros.
  4. "title": Define el título de la página web que aparece en la pestaña del navegador.
  5. "body": Define la sección del cuerpo de la página web, donde se incluyen todos los elementos que se mostrarán en la página.
  6. "h1" a "h6": Define los encabezados o títulos de diferentes niveles de jerarquía en la página web.
  7. "img": Define una imagen que se mostrará en la página web.
  8. "address": Información de contacto
  9. "br": Salto de línea sencillo
  10. "div": Etiquetas de texto HTML División
  11. "a": Define un enlace que el usuario puede hacer clic para ir a otra página web o a una sección diferente de la misma página.
  12. "p": Etiqueta de párrafo "p": Define un párrafo de texto
  13. "b": Etiqueta de negrita "b": Esta etiqueta dará formato de negrita a cualquier texto que se encuentre entre la etiqueta "b"
  14. "i": Etiqueta de cursiva Esta etiqueta formateará como cursiva cualquier texto entre la etiqueta de apertura
  15. "u"Etiqueta de subrayado: Esta etiqueta formateará cualquier texto entre la etiqueta
  16. em Etiquetas de texto HTML Énfasis
  17. strong Etiquetas de texto HTML Strong
  18. blockquote Etiquetas de texto HTML Contenido de otra fuente
  19. cite Etiquetas de texto HTML Título del libro, trabajo o web
  20. q Etiquetas de texto HTML Cita online
  21. code Etiquetas de texto HTML Visualizar una parte del código de programación
  22. code Etiquetas de texto HTML Muestra de salida de un programa informático
